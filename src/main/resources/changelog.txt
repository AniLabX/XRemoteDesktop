Версия 3.0
Новое:
+ улучшение качества воспроизведения HLS потоков в MPV посредством кеширования
+ поддержка пользовательских аргументов плеера
+ возможность установить название приложения для сервиса обнаружения

Изменения:
~ изменение пакета приложения
~ повышение версии JavaFX до 22
~ версия XRemote API повышена до V10 добавляя поддержку AniLabX 5.0+ и AniLabX MAX 1.0+

Исправления:
- исправление генерации QR кодов подключения

Версия 2.0
- Полный редизайн и рефакторинг
- Поддержка Английского, Русского и Украинского языков интерфейса с возможностью их смены
- Поддержка автостарта сервера после запуска
- Поддержка проверки обновлений
- Реализация XRemotexDiscovery сервиса для легкого подключения к серверу клиентами (на основе протокола Bonjour)
- Удален VLCj
- Поддержка MPV вместо VLC
- Поддержка автоматического скачивания MPV на Windows системах и инструкции по легкой установке для Linux/macOS
- Поддержка XRemotexRemote для удаленного управления плеером
- Расширенные настройки плеера
